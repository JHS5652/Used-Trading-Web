# 학생 대여 플랫폼
프로젝트 소개: 대학교 학생들을 위한 중고 물품 공유 플랫폼

## 📋 주요 기능 소개
- 사진을 첨부하여 물품을 추가하고 삭제할 수 있습니다

- 관리자 페이지를 통해서 회원의 정보를 관리할 수 있습니다.
  
- 회원은 자신이 등록한 물품들을 관리할 수 있습니다.
